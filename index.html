Profile

プロフィール

米永 憲司（よねなが けんじ）は、食品業界を中心に30年以上のキャリアを持つ経営プロフェッショナルです。 丸大食品での研究開発・米国事業運営から始まり、スイス系ベーカリー ARYZTA では売上を230%成長（13.6億円→31.5億円）に導きました。

フランス系ベーカリー DELIFRANCE の日本法人カントリーマネージャー、ワールド・トレーディングのCOO、 ロサンゼルスの食品商社 Poseidon Group VP など、国内外で経営の最前線を歩んできました。

現在は有限会社CBC代表取締役として独立。福井県高浜町の地域再生マネージャーをはじめ、 和歌山県みなべ町でのWeb3コミュニティ創出、企業向け経営コンサルティング、ギャラップ認定ストレングスコーチとしての組織開発を手がけています。

Projects

主な実績・プロジェクト

「描いたビジョンは即日実行」——現場に根を下ろし、データと行動力で地域の未来を切り拓いてきた実績。

01
📍 福井県 高浜町

水産加工場 HACCP認証・事業戦略構築

地域再生マネージャーとして、UMIKARA・はもと加工場のHACCP（FDA基準）認証体制の構築を主導。 水産庁「沿構R8浜活事業」の補助金申請では、設備導入による人件費削減効果の分析や費用便益計算など、 高度な財務データ立案を実施。大阪・関西万博での食育プレゼンテーション英文パネル作成もサポート。

✓ HACCP（FDA基準）認証 ✓ 水産庁補助金申請 ✓ 関西万博PR支援
02
📍 和歌山県 みなべ町

Web3 × 梅農家コミュニティ創出

伝統産業である梅農家の支援として「TUNAGUみなべ2025 梅主オーナーコレクション」NFTを企画・販売。 梅干し作り、剪定、炭焼きなどリアルな現場体験を実施し、 活動翌日には高品質動画を制作・発信。デジタルとアナログを融合した関係人口コミュニティを形成。

✓ NFT企画・販売 ✓ 現地体験プログラム ✓ 動画コンテンツ制作
03
📍 神奈川県 鷺沼

老舗ベーカリー FUJIMORI 経営業務改善

銀座発祥の老舗ベーカリー「フジモリ」の経営業務改善計画を受託。 製造原価の分析、シフト管理、バックオフィス業務の整理、オンライン出荷の改善など、 現場目線で経営基盤の立て直しに伴走。

✓ 製造原価分析 ✓ オペレーション改善 ✓ 経営基盤強化
